# Installation

## Requirements

- [Python 3.10+](https://www.python.org/downloads/)

## Install from PyPI (Recommended)

The recommended way to install Image Matching WebUI is via PyPI:

```bash
pip install imcui
```

Optional: Install with auto-download support for example datasets:

```bash
pip install imcui[datasets]
```

<Alert type="info">
<strong>Note:</strong> Example datasets (82MB) are <strong>automatically downloaded</strong> from HuggingFace on first run to your user cache directory:

- <strong>Linux/macOS</strong>: <code>~/.cache/imcui/datasets/</code>
- <strong>Windows</strong>: <code>%LOCALAPPDATA%\imcui\datasets\</code>

The download happens transparently on first launch. For offline use or custom data paths, use:

- Environment variable: <code>export IMCUI_DATA_DIR=/path/to/datasets</code>
- CLI flag: <code>imcui -d /path/to/datasets</code>
</Alert>

## Install from Source

To install from source:

```bash
git clone https://github.com/Vincentqyw/image-matching-webui.git
cd image-matching-webui
pip install -e .
```

Datasets will be auto-downloaded on first run.

## Docker Installation

Using Docker Compose:

```bash
docker pull vincentqin/image-matching-webui:latest

# Start the WebUI service
docker-compose up webui

# Or run in the background
docker-compose up -d webui
```

## Deprecated: Conda Installation

<Alert type="warning">
<strong>Deprecated:</strong> <code>environment.yaml</code> is deprecated. Please use pip instead.

```bash
git clone https://github.com/Vincentqyw/image-matching-webui.git
cd image-matching-webui
conda env create -f environment.yaml
conda activate imcui
pip install -e .
```
</Alert>

## Verification

To verify your installation:

```bash
# Check version
imcui --version

# Display help
imcui --help
```

If the installation was successful, you should see the version information and help text.
